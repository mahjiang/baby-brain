## Current Priorities (Week ~20)

1. **Start the registry** — See [registry strategy](02-gear/registry-strategy.md) and [essentials list](02-gear/essentials.md)
2. **Research childcare** — Waitlists can be 6-12+ months. See [decision framework](05-childcare/decision-framework.md)
3. **Plan the nursery nook** — See [nursery nook](03-space/nursery-nook.md)

## How This Works

- **[timeline.md](timeline.md)** is the master checklist — check it weekly for what's due now
- Each gear category has its own research file in [02-gear/](02-gear/)
- Big decisions use the [decision template](09-decisions/_template.md): research → discuss → decide
- All research is evidence-based with cited sources
- Files are meant to be updated over time — append, don't overwrite
