# Gear — Overview

## Philosophy

- **New for safety-critical items:** car seat, crib/bassinet, mattress
- **Used OK for:** clothes, toys, carriers (if structurally sound), books
- **Evidence over hype:** Ignore marketing; focus on safety ratings and practical reviews
- **Small-space friendly:** Every item earns its footprint
- **Buy-as-needed:** Don't stockpile. You'll learn what your baby actually uses.

## Buy-New vs Used Guidelines

| Buy New | Used OK | Skip Entirely |
|---------|---------|---------------|
| Car seat (no exceptions) | Clothes | Wipe warmer |
| Crib/bassinet | Toys | Bottle warmer |
| Mattress | Books | Baby shoes |
| Breast pump (insurance) | Carriers (check recalls) | Crib bedding sets |
| | Stroller (check recalls) | Baby food maker |

*Source: [CPSC guidance on used baby products](https://www.cpsc.gov/Safety-Education/Safety-Guides/Kids-and-Babies)*
