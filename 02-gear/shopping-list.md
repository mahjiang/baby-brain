# Shopping List — Buy (Don't Register)

## Buy by Week 34-36 (Before Baby)

### Sleep Accessories

- [ ] **SNOO extra sheets** (2-3) — Happiest Baby SNOO fitted sheets — ~$65-105 — *Happiest Baby site*
- [ ] **Waterproof mattress pad** for Newton 2-Stage — ~$15-25 — *Amazon / Target* — Newton has no built-in waterproof cover
- [ ] **Cat safety mesh cover** — breathable crib tent (check CPSC recalls first) — ~$30-40 — *Amazon*

### Health & First Aid

- [ ] **Pacifiers** — Philips Avent Soothie + Dr. Brown's HappyPaci — ~$12 — *Amazon / Target*
- [ ] **NoseFrida SnotSucker** — Frida Baby — ~$13 — *Amazon / Target*

- [ ] **Gas drops** — Little Remedies Gas Relief — ~$8-10 — *Amazon / Target*
- [ ] **Saline drops** — Little Remedies Saline Spray/Drops — ~$5-6 — *Amazon / Target*
- [ ] **Petroleum jelly** — Vaseline — ~$4-5 — *Grocery / Target*
