# Postpartum Plan — First Weeks

> **Plan by:** Week 30-34

## Support System

| Person | Role | Availability | Notes |
|--------|------|-------------|-------|
| | | | |
| | | | |

- [ ] Discuss who visits and when (set boundaries early)
- [ ] Identify someone who can help with meals, errands, or cat care
- [ ] Share leave plans with family — see [leave plan](../08-parental-leave/leave-plan.md)

## Meal Prep

- [ ] Start freezer meals by week 34 (soups, casseroles, burritos)
- [ ] Set up a meal train (MealTrain.com or Lotsa Helping Hands)
- [ ] Stock pantry with easy snacks (granola bars, trail mix, instant oatmeal)
- [ ] Identify local delivery options (grocery, meal kits, restaurant favorites)

## Postpartum Recovery Supplies

- [ ] Heavy pads (or hospital provides)
- [ ] Peri bottle (hospital usually provides)
- [ ] Stool softener
- [ ] Nipple cream (if breastfeeding)
- [ ] Comfortable postpartum underwear
- [ ] Ice packs / witch hazel pads (Tucks)
- [ ] Ibuprofen / acetaminophen

## Mental Health

- Know the signs of postpartum depression/anxiety (both parents can experience this)
- Postpartum Support International helpline: 1-800-944-4773
- Discuss warning signs with partner in advance
- Identify a therapist who specializes in perinatal mental health (have the number ready)
- [ ] Research local perinatal mental health resources

## Practical Prep

- [ ] Deep clean apartment (or hire cleaners) around week 36
- [ ] Stock up on household supplies (toilet paper, paper towels, trash bags)
- [ ] Set up diaper station(s)
- [ ] Program important numbers in phone (pediatrician, L&D, lactation consultant)
- [ ] Set up online grocery delivery account
- [ ] Plan pet care for hospital stay — see [cat prep](../03-space/cat-prep.md)
