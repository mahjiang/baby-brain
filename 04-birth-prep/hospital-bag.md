# Hospital Bag Packing Lists

> **Pack by:** Week 32-34

## Birthing Parent

### Documents
- [ ] ID and insurance cards
- [ ] Hospital pre-registration confirmation
- [ ] Birth preferences (printed) — see [birth preferences](birth-preferences.md)
- [ ] List of medications and allergies

### Comfort
- [ ] Robe (open front for skin-to-skin and nursing)
- [ ] Slippers with grip / flip flops for shower
- [ ] Lip balm (hospitals are dry)
- [ ] Hair ties / headband
- [ ] Pillow from home (use a non-white case so it doesn't get mixed up)
- [ ] Phone + charger + long cord
- [ ] Snacks and drinks (for labor and recovery)

### Postpartum
- [ ] Going-home outfit (loose, comfortable, maternity clothes still)
- [ ] Nursing bra / comfortable bra
- [ ] Underwear you don't care about (or use hospital mesh underwear)
- [ ] Toiletries (toothbrush, face wash, etc.)
- [ ] Nipple cream (if breastfeeding)

### Hospital Usually Provides
- Pads, mesh underwear, peri bottle
- Diapers and wipes for baby
- Swaddle blankets
- Basic toiletries (but yours are nicer)

## Partner

- [ ] Change of clothes (1-2 days' worth)
- [ ] Toiletries
- [ ] Phone + charger
- [ ] Snacks (seriously, lots of snacks)
- [ ] Entertainment (book, tablet) — labor can be long
- [ ] Pillow / blanket (partner sleeping arrangements vary by hospital)
- [ ] Cash/card for cafeteria and parking
- [ ] Camera (if not using phone)
- [ ] Comfort items for birthing parent (massage ball, essential oils if wanted)

## Baby

- [ ] Going-home outfit (NB or 0-3 month, zip-up)
- [ ] Car seat (installed in car) — see [car seat](../02-gear/car-seat.md)
- [ ] Blanket for car ride
- [ ] Hat (optional, depending on season)
