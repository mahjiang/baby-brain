# Return to Work

> Plan for this during leave, not the week before.

## Transition Planning

- [ ] Discuss expectations with manager before leave ends
- [ ] Ramp-up plan (phased return if possible?)
- [ ] Pumping at work plan (if applicable — employer must provide time and space per federal law)
- [ ] Confirm childcare start date aligns with return date
- [ ] Practice the morning routine before the first day back

## Pumping at Work

- Federal law (PUMP Act, 2022) requires employers to provide:
  - Reasonable break time to pump
  - A private space (not a bathroom) with a lock
- [ ] Identify pumping room at workplace
- [ ] Get a second pump set for work (to avoid transporting daily)
- [ ] Practice pump routine

## Emotional Prep

- Returning to work after leave is an adjustment for everyone
- It's normal to feel conflicted
- Give it at least 2-4 weeks before deciding it "isn't working"
- Check in with partner regularly about how it's going
