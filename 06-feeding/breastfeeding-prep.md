# Breastfeeding Prep

> **When:** Start learning now; practical experience begins after birth

## What to Know Before Birth

- Breastfeeding is natural but not always easy — a learning curve is normal
- Most challenges are solvable with support (lactation consultant, peer support)
- AAP recommends exclusive breastfeeding for ~6 months, continued alongside solids for 1-2+ years
- Fed is best — any amount of breastfeeding has benefits; formula is a perfectly good alternative

## Supplies

| Item | When to Buy | Notes |
|------|------------|-------|
| Nursing pillow | Before birth | Boppy or My Brest Friend |
| Nursing bras (2-3) | Week 36+ | Wait until closer to due date; size changes |
| Breast pads | Before birth | Disposable or reusable |
| Nipple cream | Before birth | Lansinoh lanolin or similar |
| Haakaa / manual pump | Before birth | Catches letdown on opposite side; saves milk |
| Breast pump | Before birth | Usually covered by insurance — see [pumping](pumping.md) |
| Milk storage bags | Before birth | If planning to pump |
| Nursing cover (optional) | Whenever | Some people use, some don't; personal preference |

## Lactation Consultant

- [ ] Check if hospital has LCs on staff (most do)
- [ ] Find an outpatient IBCLC (International Board Certified Lactation Consultant) as backup
- [ ] Check if insurance covers LC visits
