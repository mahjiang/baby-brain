# Timeline — Week-by-Week Checklist

> **How to use this:** Check your current week. Do those things. Ignore everything else until it's time. This is the antidote to overwhelm.

**Current week:** ~20 (mid-second trimester) | **Due date:** ~July 2026

## After Birth — First Month

### Day 1–3 (Hospital)
- [ ] First pediatrician visit (usually in hospital)
- [ ] Establish feeding (breastfeeding support if needed — ask for lactation consultant)
- [ ] Learn newborn basics from nurses: diapering, swaddling, cord care
- [ ] Start [weekly log](01-pregnancy/weekly-log.md) → transition to baby log
- [ ] File for birth certificate (usually offered at hospital)

### Week 1 (Home)
- [ ] **Pediatrician visit** — Usually 3-5 days after birth
- [ ] File for Social Security number (if not done at hospital)
- [ ] Add baby to health insurance (most plans give you 30 days)
- [ ] Begin tracking feeds and diapers (app or paper)
- [ ] Accept all help offered. Say yes to food deliveries.
- [ ] Sleep when baby sleeps (cliché because it's true)

### Week 2–4
- [ ] Pediatrician follow-up (~2 weeks)
- [ ] Start [milestones](07-health/milestones.md) tracking
- [ ] Begin thinking about routine (flexible, not rigid)
- [ ] Thank-you notes for gifts (or don't — it's fine)
- [ ] Review and update [finances](08-parental-leave/finances.md) — actual vs planned

## Notes
- Timing is approximate — every pregnancy is different
- Items in **bold** are highest priority for that window
- Check [complications](01-pregnancy/complications.md) for any timeline adjustments based on your situation
- This timeline assumes a term delivery (~37-40 weeks); adjust if provider advises earlier preparation
